# Python-Turtle_Crossing_Game

![turtle_img](https://i.postimg.cc/B6YFSyT6/GH9.jpg)

Image Source: [Texa Parks & Wildlife](https://tpwd.texas.gov/state-parks/martin-dies-jr/ranger-notes/fy-2021/why-did-the-turtle-cross-the-road)

This is a game implemented using Python Turtle Graphics. The game is 2D but the movement control 1D. The rectangular multi-colored cars spawn randomly on road and you as a player has to help the turtle cross the road in each level. The turtle mentioned can be controlled in only one direction upwards. Upon successful road crossing, the player graduates to next leve and the turtle respawns in same location. As the player graduates to next level, the vehicle speed also gradually increases. Happy turtle crossing!
